# MEDIATOR – termin wykonania zadania 14 stycznia 2022
Zadanie
Na bazie wzorca Mediator zaprojektuj system obslugujacy zamowienia Order.
System powinien zawierac nastepujace klasy

1. Klase odbierajaca zamowienia od klienta (zamowienie powinno przejsc w stan Received)
2. Klase weryfikujaca czy zamowienie jest poprawne (zamowienie powinno przejsc w stan
   Accepted lub Rejected)
3. Klase wykonujaca zamowienie (zamowienie powinno przejsc w stan Completed).
4. Klase logujaca wszystkie zdarzenia do pliku
5. Klase logujaca wszystkie zdarzenia do bazy (w tym przypadku na tylko udawac).
